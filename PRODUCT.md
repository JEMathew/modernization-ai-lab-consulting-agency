# Modernization AI Lab

Modernization AI Lab is an AI modernization agency.

## MVP story

A fictional aerospace manufacturer submits a portfolio of legacy data
platforms. The agency:

1. discovers the portfolio,
2. assesses business and technical health,
3. recommends a 6R disposition,
4. prioritizes modernization candidates,
5. creates modernization waves,
6. selects one Oracle data warehouse,
7. generates an Oracle-to-BigQuery engineering package,
8. validates the package,
9. requests executive approval,
10. replans when the budget or downtime constraint changes.

## Demo organization

Apex Aerospace Manufacturing.

All organizations, data, platform volumes, costs, risks, and recommendations
are synthetic and created solely for demonstration.

## MVP screens

1. Engagement Intake
2. Portfolio Assessment
3. Candidate Modernization
4. Migration Package
5. Executive Review

## Tools

- Hermes: modernization director and manager agent
- OpenAI API: reasoning, explanations, and summaries
- Codex: application engineering and migration artifact generation
- Python: deterministic scoring, ranking, 6R, validation, and packaging
- Streamlit: user interface
- SQLite: persisted projects and runs
